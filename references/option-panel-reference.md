## 面板模块速查

### 背景图片选项
- 4 种背景来源：AI 生成 / 纯设计 / 上传 / 粘贴截图
- 选择非"纯设计"选项后，显示缩放控制 + 透明度 + 混合模式面板
- 选择"纯设计封面"时隐藏所有混合模式层

### 画布尺寸切换
- 5 种比例：3:4(540×720) / 1:1(600×600) / 9:16(540×960) / 4:3(720×540) / 16:9(960×540)
- 显示为原始尺寸，导出时自动 2 倍像素
- 尺寸切换自动触发标题自适应和装饰元素重定位

### 主题模式切换
- 浅色 / 暗色两种模式
- v15.0 暗色模式完整覆盖：option-card、anchor-btn、upload-label、blend-control、fine-tune-control、watermark-control、shortcut-hint 等
- 通过 `body.dark-mode` + CSS 变量 `--MODE_*` 全局控制

### 背景缩放控制
- 5 个缩放基点：左上/右上/左下/右下/中心
- 缩放度滑块：50-200%，默认 100%

### 前景透明度
- 控制背景图层整体不透明度，范围 0-80%，默认 25%

### 混合模式增强面板
- **9 个滑块**：背景对比度(50-200%) / 亮度(50-150%) / 饱和度(0-200%) / LOGO区保护(0-100%) / 柔光叠加(0-100%) / 变暗/变亮/正片叠底/颜色混合(各 0-100%)
- **6 个预设**：清晰 / 深邃 / 柔和 / 鲜明 / 暖调 / 重置
- **保存/加载**：自定义预设通过 localStorage 持久化

- **v15.0 字体系统**：
  - "标题/副标题字体"输入框作用于 `.main-title` + `.subtitle`（默认：思源宋体）
  - "其他内容字体"输入框作用于 `.description` + `.supplement` + `.author-line` + `.meta-tag` + `.tag-label` + `.account-name-top`（默认：微软雅黑）
  - 输入 PC 端已安装字体名称 → 点击"应用"或按 Enter → 即时生效
- **重置按钮**：一键恢复所有控件到默认值（v15.0 使用 `_setSlider` / `_setSliderThen` 辅助函数精简代码）

### LOGO 显示选项
- 显示/隐藏小 LOGO，仅控制小 LOGO，不影响水印平铺
- 隐藏时同步隐藏 LOGO 区保护（blend-logo-guard）

### LOGO 布局模板
- 4 种布局：右下角（默认）/ 左下角 / 底部居中 / 顶部居中
- v15.0 修复居中布局：`left: 50%` + `translateX(-50%)` 正确居中
- 水印平铺可与任意小 LOGO 位置同时使用
- LOGO 区保护自动跟随小 LOGO 位置

### 自定义 LOGO 上传
- 支持上传任意图片作为 LOGO
- 上传后自动替换默认线稿 LOGO，保持当前布局
- "恢复默认"按钮还原为线稿 LOGO

### 水印平铺开关
- 独立开关，与小 LOGO 显示/隐藏完全解耦
- 开启时展开水印参数面板

### 水印参数面板
| 滑块 | 范围 | 默认 |
|------|------|------|
| 水印数量 | 1-16 | 1 |
| 排列列数 | 1-5 | 1 |
| 水印间距 | 0-200px | 80 |
| 水印尺寸 | 50-300px | 200 |
| 旋转角度 | -45°~45° | 0 |
| 水印透明度 | 1%-15% | 3% |

- 水印参数仅影响水印网格，不干扰小 LOGO
- 网格根据数量、列数、间距自动排列

### 导出格式
- PNG（无损）/ JPG（95%）/ WebP（92%）
- 切换时更新导出按钮文案
- 文件名自动带时间戳：`封面-{标题}-YYYYMMDD.{格式}`

## 面板交互行为总览

- 选择含背景图的选项 → 显示缩放控制 + 透明度 + 混合模式面板
- 选择"纯设计封面" → 隐藏所有混合模式层和面板
- 面板不参与导出
- 尺寸切换实时生效，自动触发标题自适应和装饰元素重定位
- 主题切换影响整体页面背景和面板颜色（v15.0 暗色模式完整覆盖所有面板元素）
- 风格微调面板始终可见，重置按钮一键恢复默认值
- 导出格式切换更新导出按钮文案和文件后缀
- LOGO 显示/隐藏仅控制小 LOGO，不影响水印
- LOGO 布局切换实时生效，LOGO 区保护自动跟随
- 水印平铺独立开关，可与任意小 LOGO 位置同时使用
- 开启水印时自动展开水印控制面板
- 水印网格根据参数自动排列，实时更新
- LOGO 区保护自动跟随小 LOGO 位置和可见性
- 标签居中排列，带菱形引导图标和底部下划线装饰
- 空间充裕时标签自动上移为底部 LOGO 留空间（`adjustBottomRowSpacing`）
- 上传自定义 LOGO 后自动替换默认线稿 LOGO
- 自定义预设通过 localStorage 跨会话持久化
- 键盘快捷键：Ctrl+E 导出 / Ctrl+S 保存预设 / 1-5 切换尺寸
- v15.0 新增 `showToast(msg, duration)` 统一 toast 提示管理
- v15.0 DOM 查询缓存：`mainTitle`, `subtitle`, `contentLayer`, `bottomRow`, `metaTags`, `toastEl` 顶部缓存
