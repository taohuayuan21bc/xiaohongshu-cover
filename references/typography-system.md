## 三、文字特效规范

### 3.1 描边效果（Text Stroke）

仅用于主标题关键词高亮 `.highlight`。

```css
.highlight {
  -webkit-text-stroke: 1px var(--ACCENT);
  text-stroke: 1px var(--ACCENT);
  color: transparent;
  /* 或描边+填充组合 */
}
```

**适用场景**：
- 深色背景 + 浅色 accent 描边
- 极简风格关键词强调
- 禁用场景：大量文字描边（破坏可读性）、手写风格

**风格关联**：#1, #2, #6（深色背景描边）、#11, #26（粗体描边强调）

### 3.2 渐变文字（Gradient Text）

用于主标题关键词 `.highlight-grad`。

```css
.highlight-grad {
  background: linear-gradient(135deg, var(--ACCENT), color-mix(in srgb, var(--ACCENT) 70%, white));
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
```

**适用场景**：
- 高端质感风格
- 科技/未来风格
- 禁用场景：纯黑字体 #6（渐变在暗底不明显）

**风格关联**：#15, #16, #18, #22, #25

### 3.3 投影效果（Text Shadow）

用于主标题/副标题增加层次感。

```css
.text-shadow {
  text-shadow: 0 2px 8px rgba(var(--TEXT_R), var(--TEXT_G), var(--TEXT_B), 0.3);
}
```

**适用场景**：
- 浅色背景增加文字层次
- 禁用场景：暗色背景（投影不可见）、小字号文字（模糊）

**风格关联**：#1, #5, #7, #12, #26, #30

### 3.4 浮雕/凹刻效果（Emboss/Engrave）

仅在极少数风格中使用。

```css
/* 浮雕（文字浮起） */
.emboss {
  text-shadow: 
    1px 1px 0 rgba(255,255,255,0.5),
    -1px -1px 0 rgba(0,0,0,0.2);
}
/* 凹刻（文字下沉） */
.engrave {
  text-shadow: 
    -1px -1px 0 rgba(255,255,255,0.3),
    1px 1px 0 rgba(0,0,0,0.3);
}
```

**适用场景**：
- 巴洛克/维多利亚风格（浮雕）
- 日式间美学（凹刻）
- 禁用场景：标题字号 < 40px（效果不明显）

**风格关联**：#15, #18（浮雕）、#24（凹刻）

### 3.5 肌理文字（Texture Text）

通过 `mask-image` 或背景叠加实现。

```css
.texture-text {
  background: url('texture-pattern.svg') repeat;
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

**适用场景**：
- 羊皮纸/旧书质感（#1 #2 #12）
- 金属/鎏金质感（#15 #18）
- 禁用场景：字号 < 30px（纹理不可见）、像素风格

**风格关联**：#1, #2, #12, #15, #18, #19

## 四、两场景文字适配

### 4.1 AI 生图场景

文字作为 AI prompt 指引的一部分，需在 prompt 中描述文字风格：

```
[模板]风格排版：大字标题型，居中布局，深色背景白色粗体大字
[字体]思源黑体 Heavy，字重 900，字间距 4px
[特效]白色文字，浅色投影，关键词亮金渐变
[字号层级]主标题 72px / 副标题 22px / 标签 12px
```

### 4.2 平面设计落地场景

文字样式直接映射至本地设计工具（PS/AI/Figma）：

| CSS 属性 | Figma 属性 | Adobe AI 属性 | 备注 |
|---------|-----------|--------------|------|
| font-family | Text → Font Family | 字符 → 字体 | 需 PC 端已安装 |
| font-weight | Text → Weight | 字符 → 字重 | 需字体支持 |
| font-size | Text → Font Size | 字符 → 字号 | 直接映射 |
| letter-spacing | Text → Letter Spacing | 字符 → 字距 | 需 /1000 换算 |
| line-height | Text → Line Height | 字符 → 行距 | 百分比转 px |
| text-stroke | Fill → Stroke | 外观 → 描边 | CSS 转 pt |
| text-shadow | Effects → Drop Shadow | 效果 → 投影 | 参数对应 |
