## 一、新增排版模板（6 套）

### T1 大字标题型
- **设计理念**：超大字号标题作为绝对视觉重心，弱化装饰，强调文字冲击力
- **布局特征**：主标题占画面 70%+ 面积，字号 60-72px（3:4 基准），字重 900
- **适用内容**：金句语录、情感共鸣、知识干货、爆款标题
- **兼容风格**：全 31 种风格，尤其适配 #26/#29/#30 等大字风格
- **HTML 适配**：`.main-title` 字号+20%，`.subtitle` 字号-10%，移除 `.description`/`.supplement`
- **元素密度**：3-5 种装饰元素（6-10 实例）

### T2 左图右文型
- **设计理念**：图片与文字并重，左侧展示视觉素材，右侧承载核心文案
- **布局特征**：新增 `.split-layout` flex 容器，左侧 45% 图片区，右侧 55% 文字区
- **适用内容**：产品种草、穿搭美妆、书评（有封面图场景）
- **兼容风格**：#7/#9/#12/#14/#17/#21/#29/#30
- **HTML 适配**：`.split-layout` 包裹原 `.content-layer`，`.image-zone` 替代部分装饰
- **元素密度**：4-6 种装饰元素（8-12 实例）

### T3 上下分栏型
- **设计理念**：明确上下分区，适合层次分明的内容
- **布局特征**：新增 `.top-section`（标题 40-50%）+ `.bottom-section`（内容 50-60%），粗分割线
- **适用内容**：知识干货、教程、清单类、问题+答案结构
- **兼容风格**：#7/#9/#10/#12/#25/#30
- **HTML 适配**：原有 content-layer 拆分为两个 section，`.section-divider` 替换 `.divider`
- **元素密度**：4-6 种装饰元素（8-12 实例）

### T4 卡片拼接型
- **设计理念**：多个独立卡片错位排列，信息点直观展示
- **布局特征**：新增 `.card-grid` CSS Grid 容器，2×2 网格，每卡片含编号+关键词+说明
- **适用内容**：合集推荐、多品对比、攻略类
- **兼容风格**：#7/#9/#11/#13/#21/#23
- **HTML 适配**：`.card-grid` 替代 `.description` 区域，主标题缩小为顶部引导语
- **元素密度**：3-4 种装饰元素（6-8 实例）

### T5 留白极简型
- **设计理念**：极简主义，大量负空间，文字精炼
- **布局特征**：仅保留主标题+副标题+标签，留白 ≥60%，标题偏上 10%
- **适用内容**：生活美学、禅意、高端品牌、文艺类
- **兼容风格**：#1/#5/#8/#14/#17/#24/#28
- **HTML 适配**：移除 `.description`/`.supplement`/`.divider`，大幅减少内容层元素
- **元素密度**：1-3 种装饰元素（2-5 实例）

### T6 新中式禅意型
- **设计理念**：东方美学，不对称构图，水墨元素
- **布局特征**：可竖排文字（`writing-mode: vertical-rl`），右上印章装饰，水墨分割线
- **适用内容**：国学、茶道、书法、东方美学
- **兼容风格**：#1/#8/#14/#18/#24/#27/#28
- **HTML 适配**：`.main-title-block` 添加竖排支持，`.seal-mark` 印章元素
- **元素密度**：5-7 种装饰元素（10-15 实例），富含国风纹样

## 二、新增文字样式体系

### 2.1 字体三分类

| 分类 | 用途 | 中文推荐 | 风格映射 |
|------|------|---------|---------|
| 标题体 | `.main-title` | 思源宋体/思源黑体/楷体/微软雅黑 | 按风格编号自动匹配 |
| 正文体 | `.subtitle`/`.description` 等 | 微软雅黑（默认）/思源宋体/楷体 | 文学类用宋体，国风用楷体 |
| 装饰体 | 印章/章节符号/科技标注 | 华文隶书/Press Start 2P/Cinzel/Orbitron | 仅在特定风格启用 |

### 2.2 七级文字层级

| L1 | `.main-title` | 48-72px | 700-900 | 主标题 |
|---|---|---|---|---|
| L2 | `.subtitle` | 20-28px | 300-600 | 副标题 |
| L3a | `.supplement` | 14-18px | 300-500 | 补充说明 |
| L3b | `.description` | 13-16px | 300-400 | 描述文字 |
| L4 | `.meta-tag` | 11-14px | 400-500 | 底部标签 |
| L5 | `.tag-label` | 11-13px | 400 | 分类标签 |
| L6 | `.account-name-top` | 11-12px | 400 | 账号名 |
| L7 | `.author-line` | 14-16px | 300-400 | 作者行 |

### 2.3 五种文字特效

| 特效 | CSS 实现 | 适用风格 | 禁用场景 |
|------|---------|---------|---------|
| 描边 | `-webkit-text-stroke` | #1/#2/#6/#11/#26 | 手写风格/大量文字 |
| 渐变 | `background-clip: text` | #15/#16/#18/#22/#25 | 暗色背景渐变不明显 |
| 投影 | `text-shadow` | #1/#5/#7/#12/#23/#26/#28/#29/#30 | 暗色背景/小字号 |
| 浮雕/凹刻 | 双向 text-shadow | #15/#18（浮雕）/#24（凹刻） | 字号 <40px |
| 肌理 | `background-clip: text` + 纹理图 | #1/#2/#12/#15/#18/#19 | 字号 <30px |

### 2.4 跨尺寸自适应

以 3:4（720px 高）为基准，按画布高度等比缩放：
- 1:1（600px）：×0.83
- 9:16（960px）：×1.33
- 4:3/16:9（540px）：×0.75

## 三、新增视觉元素（7 大类 60+ 元素）

### 3.1 肌理纹理（7 种）
P1 宣纸 / P2 羊皮纸 / P3 帆布 / P4 大理石 / P5 噪点 / P6 水彩纸 / P7 金属拉丝

### 3.2 装饰线条（7 种）
L1 古典边框 / L2 瑞士网格 / L3 手绘虚线 / L4 墨迹分割 / L5 霓虹灯线 / L6 铜板雕刻 / L7 波浪曲线

### 3.3 图标符号（7 种）
S1 菱形导引 / S2 章节符号 / S3 星星标记 / S4 箭头引导 / S5 水滴露珠 / S6 火花星芒 / S7 印章火漆

### 3.4 色块拼接（5 种）
B1 透明几何 / B2 拼贴纸片 / B3 色带条纹 / B4 圆角色块 / B5 半透明遮罩

### 3.5 噪点质感（3 种）
G1 细颗粒 / G2 粗颗粒 / G3 胶片颗粒

### 3.6 国风纹样（6 种）
C1 云纹 / C2 回纹 / C3 如意纹 / C4 水墨晕染 / C5 竹叶剪影 / C6 窗棂格

### 3.7 潮玩几何（6 种）
Y1 波点阵列 / Y2 锯齿边框 / Y3 霓虹圆环 / Y4 流光线条 / Y5 像素方块 / Y6 弹跳圆点

### 风格-元素匹配原则
- 古典/东方风格 → 国风纹样 C1-C6 + 肌理 P1/P2 + 线条 L1/L4/L6
- 现代/科技风格 → 潮玩几何 Y1-Y6 + 肌理 P5/P7 + 线条 L2/L5
- 治愈/文艺风格 → 轻量两者搭配 + 肌理 P6 + 线条 L3/L7
- 暗黑风格 → 禁用国风/潮玩，仅用肌理 P5 + 线条 L1
