# AI 背景图 Prompt 模板库（v21.0）

为每种设计风格提供定制化的 AI 背景图 prompt 模板，提升背景图质量一致性。

所有 prompt 统一规范：
- 尺寸：768x1024（3:4 竖版，宽×高=786432 ≤ 1048576，画布 540×720 的约 1.42 倍）
- 禁止：No text, no typography, no watermarks, no AI logos, no signature marks
- 风格：natural
- 生成数量：每次仅生成 1 张（`n=1`）

## 使用说明（v21.0）

1. 确定封面匹配的风格编号
2. 在对应风格分类中找到主 prompt 模板，直接使用
3. 使用 `image_gen` 工具生成，参数：`size="768x1024"`, `style="natural"`, `n=1`
   - `image_gen` 要求宽×高 ≤ 1024×1024，768×1024 = 786432 满足限制且保持 3:4 比例
