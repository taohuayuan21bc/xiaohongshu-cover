## 二、模板详细定义

### T1 大字标题型（Big Title）

**设计理念**：以超大字号标题作为绝对视觉重心，弱化装饰，强调文字冲击力。

**布局结构**：
```
┌──────────────────────┐
│  [账号名/标签区]       │  ← header-row（右上）
│                      │
│                      │
│    【超大主标题】      │  ← main-title（居中，占面积70%+）
│                      │
│    ── 分割线 ──       │
│    副标题/作者         │
│                      │
│  [标签1][标签2][标签3] │  ← bottom-row
└──────────────────────┘
```

**内容层 HTML 结构调整**：
- `.main-title` font-size 增至 60-72px，letter-spacing 2-4px
- `.subtitle` font-size 18-22px
- `.description` 可选省略
- `.supplement` 可选省略
- `.header-row` 中的 `.tag-row` 左移，`account-name-top` 右上

**尺寸适配规则**：
- 3:4（540×720）：主标题 60-72px，副标题 18-22px
- 1:1（600×600）：主标题 52-64px，副标题 16-20px
- 9:16（540×960）：主标题 56-68px，副标题 18-22px
- 4:3（720×540）：主标题 48-60px，副标题 16-18px
- 16:9（960×540）：主标题 44-56px，副标题 14-18px

**适用风格编号**：#1-#31 全兼容，尤其适配 #26（简约醒目大字）、#29（莫兰迪大字）、#30（干净蓝底大字）

---

### T2 左图右文型（Left Image Right Text）

**设计理念**：图片与文字并重，适合需要展示视觉素材的内容类型。

**布局结构**：
```
┌──────────────────────────┐
│ [账号名]                  │ ← 右上
│                          │
│ ┌──────┐                 │
│ │      │  【主标题】      │
│ │ 图片 │  ──分割线──      │
│ │ 区域 │  副标题          │
│ │      │  描述文字        │
│ └──────┘                 │
│                          │
│      [标签1][标签2][标签3] │
└──────────────────────────┘
```

**内容层 HTML 结构调整**：
- 新增 `.split-layout` 包裹层：`display: flex; gap: 24px; align-items: center;`
- 左侧 `.image-zone`：占 45% 宽度，放置 AI 背景图或色块
- 右侧 `.text-zone`：占 55% 宽度，包含标题/副标题/描述
- `.header-row` 移至文本区顶部

**尺寸适配规则**：
- 3:4：左侧图片区 200×266px，右侧文字区 padding 20px
- 1:1：左侧图片区 240×240px
- 9:16：左侧图片区 200×300px
- 4:3/16:9 横版：改为上下分栏更佳

**适用风格编号**：#7, #9, #12, #14, #17, #21, #29, #30

---

### T3 上下分栏型（Top-Bottom Split）

**设计理念**：明确上下分区，适合层次分明的内容结构，如"问题+答案""场景+方法"。

**布局结构**：
```
┌──────────────────────┐
│  [标签]  [账号名]      │
│                      │
│  ═══ 上栏标题区 ═══   │ ← 主标题+副标题，占40-50%
│                      │
│  ───── 分界线 ─────   │ ← 明确分割
│                      │
│  下栏内容区           │ ← 描述+补充+标签，占50-60%
│  详细说明文字...       │
│  ┌─┐ ┌─┐ ┌─┐        │
│  │1│ │2│ │3│        │ ← 卡片式要点
│                      │
└──────────────────────┘
```

**内容层 HTML 结构调整**：
- 新增 `.top-section` + `.bottom-section` 分区
- `.top-section`：标题+副标题+分割线
- `.bottom-section`：描述+补充+要点卡片
- 分区之间使用 `.section-divider`（粗分割线）

**尺寸适配规则**：
- 3:4：上栏 280px，下栏 440px
- 1:1：上栏 260px，下栏 340px
- 9:16：上栏 350px，下栏 610px
- 横版：改为左右分栏

**适用风格编号**：#7, #9, #10, #12, #25, #30

---

### T4 卡片拼接型（Card Collage）

**设计理念**：多个独立卡片错位排列，适合合集、多品推荐类内容。

**布局结构**：
```
┌──────────────────────┐
│ [标题区]  [账号名]     │ ← 顶部小标题
│                      │
│ ┌──────┐ ┌──────┐    │
│ │ 卡片1 │ │ 卡片2 │    │ ← 上排双卡
│ └──────┘ └──────┘    │
│     ┌──────┐         │
│     │ 卡片3 │         │ ← 下排单卡（错位）
│     └──────┘         │
│                      │
│ [标签1][标签2][标签3]  │
└──────────────────────┘
```

**内容层 HTML 结构调整**：
- 新增 `.card-grid` 网格容器：`display: grid; grid-template-columns: 1fr 1fr; gap: 8px;`
- 每个 `.info-card`：带边框+圆角+背景色块
- 卡片内包含：编号/图标 + 关键词 + 简短说明
- 主标题缩小为顶部引导语

**尺寸适配规则**：
- 3:4：卡片网格占据画面 60%，每卡约 100×80px
- 1:1：卡片各占 110×90px
- 9:16：卡片可增至 3×2 网格
- 横版：改为横向排列 4-5 卡片

**适用风格编号**：#7, #9, #11, #13, #21, #23

---

### T5 留白极简型（Minimal Whitespace）

**设计理念**：极简主义，大量留白，文字精炼，装饰几乎为零。

**布局结构**：
```
┌──────────────────────┐
│                      │
│                      │
│     【核心标题】      │ ← 居中偏上 10%
│                      │
│      —— 副标题       │
│                      │
│                      │
│                      │
│                      │
│  [标签]              │ ← 底部居中
└──────────────────────┘
```

**内容层 HTML 结构调整**：
- 大幅简化内容层：仅保留 `.main-title` + `.subtitle` + `.meta-tags`
- 移除 `.description`、`.supplement`、`.divider`
- `.main-title-block` margin-top 设为 20%
- `.meta-tags` 底部居中排列
- 装饰元素减少至 2-3 个超低透明度光晕

**尺寸适配规则**：
- 所有尺寸通用：主标题字号 48-56px，副标题 16-18px
- 留白面积始终 ≥ 60%

**适用风格编号**：#1, #5, #8, #14, #17, #24, #28

---

### T6 新中式禅意型（New Chinese Zen）

**设计理念**：融合东方美学，竖排文字可选，水墨元素，不对称构图。

**布局结构**：
```
┌──────────────────────┐
│                [印章] │ ← 右上角印章装饰
│                      │
│    ┃                │
│    ┃  【竖排】       │ ← 可选竖排标题（writing-mode: vertical-rl）
│    ┃  主标题        │
│    ┃                │
│                      │
│  ── 副标题/作者 ──   │
│                      │
│  [标签]              │ ← 左下标签
└──────────────────────┘
```

**内容层 HTML 结构调整**：
- `.main-title-block` 添加 `writing-mode: vertical-rl` 支持竖排
- 新增 `.seal-mark` 印章装饰元素
- `.divider` 替换为水墨风格分割线
- 整个 `.content-layer` 使用不对称偏移

**尺寸适配规则**：
- 3:4：竖排标题右偏移 15%，字号 48-56px
- 1:1：竖排标题居中，字号 42-48px
- 其他尺寸：建议保持 3:4 比例

**适用风格编号**：#1, #8, #14, #18, #24, #27, #28

## 三、模板与风格联动规则

| 模板 → | 优先匹配风格 | 禁配风格（风格冲突） |
|--------|------------|-------------------|
| T1 大字标题型 | #26, #29, #30, #11 | #15（巴洛克装饰过多） |
| T2 左图右文型 | #7, #12, #14, #21 | #8（水墨留白需居中） |
| T3 上下分栏型 | #7, #9, #10, #12 | #15, #18（装饰干扰分区） |
| T4 卡片拼接型 | #7, #11, #13, #21 | #1, #3（史诗/浪漫不适合卡片） |
| T5 留白极简型 | #1, #8, #17, #24 | #11, #13, #20（粗野/波普/像素冲突） |
| T6 新中式禅意型 | #8, #14, #24, #28 | #7, #11, #16, #25（现代/科技风格冲突） |

### 智能匹配推荐优先级

当内容包含以下特征时，优先推荐对应模板：
- 标题简短有力（≤8字）→ T1 或 T5
- 有产品/视觉图需求 → T2
- 内容层次分明（步骤/清单）→ T3
- 包含多个并列项 → T4
- 内容调性偏文艺/高级 → T5 或 T6
- 用户明确东方/国风主题 → T6

## 四、模板与框架注入兼容性

所有模板均通过修改 `<!-- INJECT_CANVAS_CONTENT -->` 中的 `.content-layer` 结构实现，不影响框架整体。具体适配：
- T1/T5/T6：修改 `.content-layer` 内各元素排列与字号
- T2：新增 `.split-layout` 包裹层，不再改变其他结构
- T3：新增 `.top-section` + `.bottom-section` 分区
- T4：新增 `.card-grid` 替代 `.description` 区域

框架的 4 个注入标记（INJECT_STYLE_CSS / INJECT_CANVAS_CONTENT / INJECT_AI_BG_PATHS / INJECT_INIT_BG）保持不变。
