## 四、联动逻辑说明

### 4.1 单参数全链触发

用户只需指定一个维度，系统自动推导其余：

```
用户输入："用大字标题型模板"
→ 匹配模板 T1
→ T1 默认风格：风格池全兼容
→ 根据内容品类自动匹配最佳风格
→ 风格→字体（typography-system §五）
→ 风格→元素（visual-elements-library §三）
→ 输出完整精细设计
```

### 4.2 模板-文字-元素联动链

```
模板编号 + 风格编号
  ├─ 模板 → 字号调整（typography-system §2.3）
  ├─ 模板 → 元素密度（visual-elements-library §2.3）
  ├─ 风格 → 字体+字重+特效（typography-system §五）
  ├─ 风格 → 肌理+线条+国风/潮玩（visual-elements-library §三）
  ├─ 风格 → 配色方案（design-styles-reference §二）
  └─ 风格 → AI 背景 prompt（ai-bg-prompt-templates）
```

## 六、降级与兜底策略

| 场景 | 策略 |
|------|------|
| 资源模块未加载 | 主程序内置基本匹配规则，可独立完成方案推荐输出 |
| 模板/风格无法联动 | 回退到默认 T3（上下分栏）+ 自动匹配风格 |
| 内容分析失败 | 默认归类"知识干货"，T3+#7（瑞士国际主义） |
| 超长文案 | 自动拆分 + 强制 T2/T3 适配 |
| 违规内容 | 直接拒答，不触发任何匹配 |
