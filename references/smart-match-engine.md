### 1.4 字数长度分级

| 级别 | 主标题字数 | 处理策略 |
|------|---------|---------|
| 短标题 | ≤6 字 | 优先 T1（大字标题型）或 T5（留白极简型） |
| 中等标题 | 7-12 字 | 所有模板均可 |
| 长标题 | 13-20 字 | 避免 T1，优先 T3（上下分栏）或 T2（左图右文） |
| 超长标题 | >20 字 | 仅限 T3，自动拆分主副标题，提醒用户精简 |

## 二、三套方案差异化生成规则

### 2.1 方案差异化维度

| 维度 | 方案 A（流量爆款风） | 方案 B（质感高级风） | 方案 C（小众特色风） |
|------|-------------------|-------------------|-------------------|
| **风格方向** | 醒目标题+强力视觉冲击 | 克制优雅+高级审美 | 独特性格+记忆点 |
| **模板倾向** | T1 大字标题型 / T3 上下分栏型 | T5 留白极简型 / T2 左图右文型 | T6 新中式禅意型 / T4 卡片拼接型 |
| **配色倾向** | 高对比、强 accent、暖色系 | 低饱和、莫兰迪、中性色系 | 特色配色、反差色、风格化 |
| **字体倾向** | 粗体无衬线（900 字重） | 细衬线/无衬线（400-600 字重） | 特色字体（楷体/手写/像素） |
| **装饰密度** | 高密度（10-15 实例） | 低密度（5-8 实例） | 中等密度（8-12 实例） |
| **传播目标** | 点击率、互动率 | 品牌感、调性建立 | 记忆点、差异化 |

### 2.2 方案生成流程

```
输入内容 → 
  Step 1：品类识别（确定主品类）
  Step 2：调性分析（确定内容调性）
  Step 3：受众分析（确定目标受众）
  Step 4：字数分级（确定标题长度级别）
  Step 5：三套方案生成 →
    ├─ 方案 A：匹配流量爆款风
    │   └─ 从对应风格池选 1 个风格 + 1 个模板 → 核查风格-模板联动兼容性
    ├─ 方案 B：匹配质感高级风
    │   └─ 从对应风格池选 1 个风格 + 1 个模板 → 核查风格-模板联动兼容性
    └─ 方案 C：匹配小众特色风
        └─ 从对应风格池选 1 个风格 + 1 个模板 → 核查风格-模板联动兼容性
  Step 6：三套方案差异化核查
    └─ 确保风格编号不重复、模板不重复、差异化维度各不重叠
```

### 2.3 风格池分配规则

根据识别的品类和调性，为每套方案分配候选风格池：

| 方案 | 风格池来源 | 选风规则 |
|------|----------|---------|
| A 流量爆款风 | 当前品类中高流量风格（如 #26, #29, #30, #11, #7） | 选对比度最高、accent 最醒目的 |
| B 质感高级风 | 当前品类中高质感风格（如 #14, #17, #1, #5, #9, #28） | 选低饱和、高级灰色调的 |
| C 小众特色风 | 当前品类中风格鲜明/跨品类惊艳的风格（如 #8, #16, #18, #20, #22, #24, #27） | 选有强烈记忆点的 |

## 三、模板-文字-元素联动规则

### 3.1 联动链

```
用户选定方案 → 
  Step A：确定风格编号 + 模板编号
  Step B：风格编号 → 查询 typography-system.md §五 → 确定字体/字重/特效
  Step C：模板编号 → 查询 typography-system.md §2.3 → 确定字号调整
  Step D：风格编号 + 模板编号 → 查询 visual-elements-library.md §三 → 确定元素搭配
  Step E：风格编号 → 查询 design-styles-reference.md → 确定配色方案
  Step F：生成完整精细版设计
```

### 3.2 用户指定单参数联动

| 用户输入 | 联动加载 | 跳过步骤 |
|---------|---------|---------|
| 仅指定风格（如"#7 瑞士风"） | 风格→字体+元素+配色，模板默认 T3 | 跳过方案选择 |
| 仅指定模板（如"大字标题型"） | 模板→字号+元素密度，风格智能匹配 | 跳过方案选择 |
| 指定风格+模板 | 全部联动加载 | 跳过方案选择 |
| 只提供内容（无指定） | 智能匹配→3方案→用户选择→联动加载 | 全部执行 |

## 五、边界场景匹配策略

| 场景 | 匹配策略 |
|------|---------|
| 内容品类无法识别 | 默认归类为"知识干货"，以中性风格输出 |
| 调性分析模糊 | 默认"活泼有趣"，方案 B 保留"高级质感"作为对比 |
| 品类+风格无默认映射 | 从 31 种风格中匹配最接近氛围的风格 |
| 超长标题（>20 字） | 自动拆分为主标题（前 12 字）+副标题（剩余），强制使用 T3 模板 |
| 无标题仅有描述 | 从描述中提取关键短语拼接为临时标题 |
| 违规内容 | 拒答，不进行任何匹配 |
| 用户要求切换方案 | 加载新方案对应资源，重新输出第二步 |
