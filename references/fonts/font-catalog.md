## 二、加载规则

```
用户选定风格编号 →
  Step 1：查本表确定字体分类（如 #7 → F2 无衬线现代）
  Step 2：加载对应分类文件（如 fonts/sans-modern.md）
  Step 3：从分类文件读取 CDN link + CSS font-family 声明
  Step 4：注入 INJECT_STYLE_CSS 时使用分类文件的完整内容
```

**节省原理**：
- 旧版：31 种风格各含独立 CDN link（~900 tokens）+ 分散的字体描述（~2000 tokens）= ~3000 tokens
- 新版：按需加载 1 个分类文件（~50 tokens）+ 本目录（~200 tokens）= ~250 tokens/次
- 节省率：~92%
