# 视觉设计元素库（v22.0）

> **用途**：本文件为按需加载资源。仅在用户选定方案后、需要输出精细版面设计时，由主程序唤醒加载对应风格的元素组合。首轮方案预览时无需加载。
> **触发时机**：用户选定 3 套方案中的某套后，或用户直接指定风格/模板时加载。非按需场景下优先使用装饰元素参考库原有映射。

---

## 一、元素分类总览

### 1.1 肌理纹理（Texture & Pattern）

营造纸张、布料、材质触感的背景纹理。

| 编号 | 元素名称 | CSS 实现方式 | 适用风格 | 透明度 | 叠加模式 |
|:----:|---------|------------|---------|--------|---------|
| P1 | 宣纸纹理 | `repeating-linear-gradient` + 噪点 | #1, #8, #18, #24, #28 | 0.03-0.06 | multiply |
| P2 | 羊皮纸纹理 | `radial-gradient` + 不规则边界 | #1, #2, #12, #15 | 0.04-0.08 | multiply |
| P3 | 帆布纹理 | `repeating-conic-gradient` 微网格 | #9, #11, #19 | 0.02-0.05 | overlay |
| P4 | 大理石纹理 | SVG 不规则曲线描摹 | #14, #17, #24 | 0.03-0.06 | soft-light |
| P5 | 噪点质感 | SVG feTurbulence 滤镜 | #6, #12, #20, #22, #25 | 0.02-0.08 | grain |
| P6 | 水彩纸纹理 | 不规则柔和边界 + 纤维纹理 | #5, #8, #14, #19, #31 | 0.03-0.06 | multiply |
| P7 | 金属拉丝 | 单向线性渐变 + 微噪点 | #15, #16, #18, #25 | 0.03-0.06 | overlay |

## 四、元素加载优先级

为减少运行时 token 占用，按以下优先级分步加载：

### 第一优先级（始终加载）
基础装饰元素（原有 `decor-elements-reference.md` 中的 CSS 组件 1-27 和风格映射表）

### 第二优先级（选定方案后加载）
- 选定风格对应的肌理纹理（§1.1）
- 选定模板对应的装饰线条（§1.2）
- 基本图标符号（§1.3）

### 第三优先级（精细设计中按需加载）
- 国风纹样 C1-C6（仅国风风格）
- 潮玩几何 Y1-Y6（仅现代/科技/波普风格）
- 噪点质感 G1-G3（仅暗黑/复古风格）

## 五、与 AI 生图 + 平面设计双场景适配

### 5.1 AI 生图元素 prompt 模板

```
[装饰元素] 风格：[风格名称] | 肌理：[肌理名称] | 装饰：[线条类型]+[图标类型]+[国风/潮玩]
示例："宣纸纹理底，细颗粒噪点，右上角朱砂印章，左下角水墨晕染，回纹边框装饰"
```

### 5.2 平面设计元素清单格式

```
视觉元素清单：
├─ 背景肌理：宣纸纹理（叠加模式：multiply，透明度：5%）
├─ 右上装饰：朱砂红印章 § 24px, rotate(-15deg)
├─ 左下装饰：水墨晕染 120×180px, blur(8px)
├─ 边框装饰：回纹图案 1px 边框, 圆角 4px
└─ 氛围噪点：细颗粒噪点 overlay 3%
```
