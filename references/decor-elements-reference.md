## 通用 CSS 组件

所有装饰元素放在 `<div class="decor-layer">` 内（`z-index: 1`, `pointer-events: none`）。

### v6.0 位置自适应标签

每个装饰元素应添加 `data-decor-pos` 属性，用于尺寸切换时自动重定位：

| 属性值 | 定位含义 | 基准位置（540×720） |
|--------|---------|-------------------|
| `tl` | 左上角 | top: 58px, left: 32px |
| `tr` | 右上角 | top: 72px, right: 43px |
| `bl` | 左下角 | bottom: 108px, left: 43px |
| `br` | 右下角 | bottom: 86px, right: 54px |
| `center` | 居中 | top: 50%, left: 50% |

## 装饰元素布局原则

- **左上区域**：窗棂剪影、垂直装饰线、飘散文字、章节符号、铜板雕刻线
- **右上区域**：灯笼光晕、光斑、星光、水彩晕染、羽毛轻飘
- **左下区域**：坠落花瓣、灰烬颗粒、几何线条、墨迹晕染、手绘虚线
- **右下区域**：留白给 LOGO
- **全局散布**：旧书边框、飘散文字、气泡浮动、同心涟漪
- **动画元素**：每封面 2-3 个
- **密度控制**：总计 8-15 个装饰实例

## 配色自适应

- 使用当前风格 accent 色，`rgba(ACCENT_R, ACCENT_G, ACCENT_B, 0.04-0.25)` 控制透明度
- 深色背景 → 浅 accent，透明度 0.06-0.2
- 浅色背景 → 深 accent，透明度 0.04-0.12
- 动画元素透明度浮动 ≤ 0.3

## v6.0 装饰元素位置自适应

生成 HTML 时，每个装饰元素应添加 `data-decor-pos` 属性，以便尺寸切换时通过 `repositionDecorElements()` 自动重定位：

```html
<!-- 示例：左上角光晕 -->
<div class="glow-orb" data-decor-pos="tl" style="top:58px;left:32px;"></div>

<!-- 示例：右上角星光 -->
<div class="star-particle" data-decor-pos="tr" style="top:72px;right:43px;"></div>

<!-- 示例：左下角花瓣 -->
<div class="petal" data-decor-pos="bl" style="bottom:108px;left:43px;"></div>

<!-- 示例：居中水彩 -->
<div class="watercolor-wash" data-decor-pos="center" style="top:50%;left:50%;"></div>
```

`repositionDecorElements()` 根据当前画布尺寸自动计算各位置的像素值，无需手动处理。
